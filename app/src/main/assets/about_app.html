RollingDesign 介绍

RollingDesign是一款基于设计师网站 Drrrible 提供API 开发的一款第三方Android客户端，该应用使用了MVP架构模式，并采用RxJava + Retrofit 进行网络通信。现实功能包括浏览作品、收藏、点赞、关注、查看作者信息等（上传作品功能暂不开放）。

涉及开源库

  • Retrofit HTTP网络请求框架
  • RxJava 基于事件的线程调度工具
  • Fresco 强大的图片加载库
  • Gson Json解析工具
  • ButterKnife 控件初始化精简工具
  • EventBus 事件驱动分发工具
  • BaseRecyclerViewAdapterHelper 经过封装的RecyclerView适配器
  • PhotoView 图片缩放控件

开发环境

  • AndroidStudio 2.2.2
  • Gradle version 2.14.1
  • Gradle plugin version 2.2.2
  • Build tools version 25.0.2
